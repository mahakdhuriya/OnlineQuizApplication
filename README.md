# Online Quiz Application (Java)

## Description
A console-based Java quiz application that allows users to answer multiple-choice questions and receive their score instantly.

## Features
- Multiple-choice questions
- Instant score calculation
- Percentage calculation
- Console-based interface

## Technologies Used
- Java
- VS Code
- JDK 17

## Project Structure
```
OnlineQuizApplication
│── Main.java
│── Quiz.java
│── Question.java
│── README.md
```

## How to Run

Compile:
```bash
javac *.java
```

Run:
```bash
java Main
```
